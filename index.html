O Training Mode

Visto que muitos jogos de luta não possuem um modo treino, como por exemplo Real Bout Special, Samurai Shodown, Fatal Fury 3 entre outros, mesmo na sua versão de console. Ou ainda pode ter esse modo nas suas versões de arcade.

Usando o emulador MAME-RR criei scripts escritos na linguagem lua adaptei um modo treino para esses jogos, semelhante ao dos jogos que possuem o seu modo treino originalmente.

Será disponibilizado os modos treino que consegui adaptar com uma breve descrição de como usar e link para download.

Rage of The Dragons

Modo treino adaptado para o jogo Rage of The Dragon. Usa o emulador MAME, (já incluso) para iniciar o jogo execute o arquivo start.bat.

Ao iniciar o jogo vai diretamente para a tela de seleção de personagens, poderá selecionar a dupla do player 2.

  • Marcador de combos fixo na tela
  • Marcador de danos fixo na tela
  • Marcador de max combo fixo na tela
  • Para retornar a tela de player select mantenha pressionado START por dois segundos
  • Para fechar o jogo pressione tecla ESC

Ninja Master's

Modo treino adaptado para o jogo Ninja Master’s. Usa o emulador MAME, (já incluso) para iniciar o jogo execute o arquivo INICIAR MAME.bat.

  • Ao iniciar o jogo vai diretamente para a tela de player select
  • BOSS habilitados
  • O personagem do Player 2 é selecionado automaticamente
  • Energia do player 2 é infinita
  • Player 1 sempre com energia vermelha
  • Barra de especial funciona normalmente
  • Para retornar a tela de player select, tecla L
  • Para fechar o jogo pressione tecla ESC

Fatal Fury 3 - Road The Final Victory

Modo treino adaptado para o jogo Fatal Fury 3. Usa o emulador MAME, (já incluso) para iniciar o jogo execute o arquivo start.bat.

  • Ao iniciar o jogo vai diretamente para a tela título do jogo, onde se pode jogar no modo de 1 ou 2 players
  • Pressionando botão C na tela título ativar o modo treino
  • É possível escolher o personagem para o Player 2
  • Chonrei, Chonshu e Yamazaji selecionáveis na player select
  • Marcador de combos e danos fixo na tela
  • Para retornar a tela de player select mantenha COIN / SELECT pressionado por alguns segundos
  • Para fechar o jogo pressione tecla ESC
